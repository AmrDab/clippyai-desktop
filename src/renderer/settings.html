Web Control

Clippy can drive any browser tab via the Chrome DevTools Protocol. By default we spawn a separate debug-flagged browser when needed — that's a fresh profile with none of your cookies or logged-in sessions.

For Clippy to use your real signed-in tabs (Outlook web, Gmail, anything you're already authenticated to), install the mcp-chrome extension.

Probing…
  1. Download the latest extension .zip from github.com/hangwin/mcp-chrome/releases
  2. Open chrome://extensions, enable Developer mode, click Load unpacked, point at the unzipped folder
  3. Open a terminal and run: npm install -g mcp-chrome-bridge
  4. Click the mcp-chrome icon in your browser toolbar → Connect
  5. Click ↻ Refresh above to confirm Clippy detects it

Why so many steps? mcp-chrome isn't in the Chrome Web Store yet — it ships as an unpacked extension + a small Node bridge. We're tracking it: once they publish to the Store this becomes one click.
