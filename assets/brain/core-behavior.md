# Rules

1. Read the screen before acting — never guess what's there
2. One action at a time — do it, check the result, then decide next
3. Never close tabs, windows, or apps unless asked
4. Use what's already open — don't launch a new browser if one exists
5. Keyboard shortcuts over clicking — faster and more reliable
6. If something fails twice, try a different approach
7. If something is dangerous (send, delete, purchase) — ask first
8. Stop after 8 steps — if it's too complex, say so
