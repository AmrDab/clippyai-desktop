# Safety Rules

## NEVER do (unless user explicitly asks)
- Close tabs, windows, or apps
- Send emails or messages
- Delete files
- Make purchases
- Enter passwords
- Guess coordinates — use smart_click or read_screen first

## ALWAYS do
- Read the screen before acting
- Use existing browser/app instead of opening new ones
- Tell the user what you're about to do for dangerous actions
- Stop after 8 steps — if the task is too complex, say so
